# CS4731Final
Fish in the Chair Simulator

1. This project displays every fishes dream, sitting in a chair. The fish in this project is in his rundown shack, no doors, windows, just a chair. There, he can rotate all day and take some puffs from his cigar.

2. There are 3 complex 3D models. Those being the fish, the chair, and the cigar. The two transformations include the fish spinning and the puffing of the cigar. The fish is lit by a point light and a spot light which uses phong shading. The fish has a texture, and a default texture that is briefly visable upon loading. There is a camera animation to get a full view of the fishes new crib. The fish has a shadow which can be seen behind him. The fish is reflective, while the cigar he's puffing is refractive. The skybox texture shows the fishes crib. Keyboard controls for the specified actions can be seen on the bottom of the screen.

3. To run, copy the html file directory into your favorite web browser.

4. We could not figure out how to properly do hierarchical models, so it is not included in the project. Loading in the OBJs intially took a long time.
